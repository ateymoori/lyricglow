### Quick Install

1. [Download LyricGlow.dmg](https://github.com/ateymoori/lyricglow/releases/latest)
2. Open the DMG file
3. Drag to Applications folder
4. Launch and enjoy

**Requirements:**
- macOS 11.0 or later
- 100 MB free space
- Internet connection for lyrics

### Build from Source

```bash
git clone https://github.com/ateymoori/lyricglow.git
cd lyricglow
npm install
npm start
npm run dist:mac
```
